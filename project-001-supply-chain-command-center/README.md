# Supply Chain Command Center

## Overview
An interactive executive dashboard to monitor key supply chain health metrics, bridging the gap between advanced analytics and decision-making. This project complements the Data Science Portfolio by visualizing insights from forecasting and inventory optimization models.

## Features
- **KPI Scorecard**: Real-time view of Service Level, Inventory Turnover, and Forecast Accuracy.
- **Demand vs. Forecast**: Visualization of historical sales vs. predicted demand.
- **Inventory Health**: Monitoring stock levels against safety stock thresholds.

## Tech Stack
- **Python**: Core logic
- **Streamlit**: Interactive web application framework
- **Plotly**: Advanced data visualization
- **Pandas**: Data manipulation

## Quick Start

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the application:**
    ```bash
    streamlit run app.py
    ```
